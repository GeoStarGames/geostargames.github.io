Flow25 & lib25 Documentation

Welcome to the documentation for the Flow25 programming language and its library system, lib25. This guide will walk you through the syntax, commands, and examples for both languages.

Flow25 Language

Flow25 is a simple programming language designed for creating small programs and games. Below are the available commands:

Basic Commands

  • #comp x8_mod: Must be the first line of every Flow25 program.
  • var name = value: Declares a variable (int).
  • var name %= "value": Declares a variable (string).
  • set name = value: Assigns a value to a variable.
  • set name += value: Adds a value to a variable.
  • set name -= value: Subtracts a value from a variable.
  • out value: Outputs a value to the console.
  • quit: Ends the program.

Control Flow

  • if a >= b then label: Jumps to a label if the condition is true.
  • run label: Jumps to a label.
  • call label: Jumps to a label, also storing adress in call stack.
  • return: Returns to the latest call stack adress.
  • label name: Defines a label.

GPU Commands

  • gpu_send value: Sends a value to the GPU.
  • gpu_call command: Executes a GPU command (set_x, set_y, set_char, set_color, flip, reset).

Example

Hello World

#comp x8_mod

var message %= "Hello, World!"
out message

quit

lib25 Language

lib25 is a library system for Flow25. It allows you to create reusable functions and include them in your Flow25 programs.

Creating a Library

  • #comp lib25: Must be the first line of every lib25 file.
  • create library_name: Creates a new library.
  • add function_name: Defines a new function.
  • arg name: Defines a function argument.
  • end function_name: Ends a function definition.
  • push value: Stores a value to push_data (can be read using pop).
  • pop value: Gets a value which was stored in push_data.

Example

Math Library

#comp lib25
create math_lib

add add_numbers
arg x
arg y
var result = 0
set result += x
set result += y
push result
end add_numbers

add subtract_numbers
arg x
arg y
var result = 0
set result += x
set result -= y
push result
end subtract_numbers

Using a Library

To use a library in your Flow25 program, include it with the Add Library button and call its functions with call_lib.

Using the Math Library

#comp x8_mod

var result = 0
call_lib math_lib add_numbers 5 10
pop result
out result  // Outputs 15

call_lib math_lib subtract_numbers 20 7
pop result
out result  // Outputs 13

quit

Conclusion

Flow25 and lib25 are powerful tools for creating small programs and games. With this documentation, you should be able to get started quickly. Happy coding!
